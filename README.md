# Devlopment
To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

Link bin:

```bash
bun link
```

Execute `create-hydrastack-app` locally:
You will need to run `bun link` first
```bash
create-hydrastack-app
```
